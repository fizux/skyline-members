Log Sheets

This directory includes the executable you'll need for the logsheet program, and an interface to search previously-submitted logsheets. Usually the two links you'll be using the most are:

  1. DOWNLOAD -- Download the latest logsheet program. This will have the latest information about members, temporary members, and will include the logsheets of the last 20 days of operations.
  2. UPLOAD -- Upload the logsheet of a day's operations. The program allows multiple uploading of a logsheet for a single day.

There are two versions of the ssclog program, one for Windows and one for the MAC. Both programs are compressed as zip files.

Linux users can use the windows ssclog program under WINE, it's been tested pretty thoroughly and seems to work just fine.
